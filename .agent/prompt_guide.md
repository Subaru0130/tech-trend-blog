# AIプロンプト調整ガイド：デザインのズレを防ぐために

AIに対して、より正確にデザインの意図を伝えるためのコツをまとめました。

## 1. 具体的な「視覚的指示」を含める
「いい感じにして」ではなく、具体的な要素を指定すると精度が上がります。

*   **NG**: 「背景画像を入れて」
*   **OK**: 「**抽象的な幾何学模様の**背景画像を、**不透明度20%**で、テキストの**背面に**配置して」

## 2. 「否定命令」よりも「肯定命令」を使う
「〜しないで」よりも「〜して」の方が伝わりやすい傾向があります。

*   **NG**: 「安っぽくしないで」
*   **OK**: 「**高級感のある、落ち着いた配色（白とスレートグレー）**を使って」

## 3. 参考イメージ（メタファー）を使う
具体的なデザイン用語がわからなくても、イメージを共有することで伝わります。

*   「Appleのウェブサイトのような、余白を多用したデザインで」
*   「雑誌のような、明朝体を使ったレイアウトで」

## 4. 今回のケースでの改善例

今回の「背景画像」と「CTA（他の記事を見る）」の修正を依頼する場合の理想的なプロンプト例です。

> **プロンプト例:**
> トップページのヒーローセクション（一番上の部分）の背景に、**テック感のある抽象的な画像**を追加してください。
> ただし、文字が読みにくくならないよう、**画像の不透明度は低く（薄く）**設定し、テキストの**背面**に配置してください。
> 
> また、記事下の「他の記事を見る」ボタンのエリアは、背景色が薄すぎて目立たないので、**背景色を薄い青（blue-50）**に変更し、ボタン自体は**濃い青（blue-600）**にしてコントラストを上げてください。

このように、「どこを」「どのように」「なぜ（目立たないから）」をセットで伝えると、一発で理想に近い出力が得られます。
