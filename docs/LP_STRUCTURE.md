# LP Structure: Best Buy Guide (ベストバイガイド)

Based on the "Muscle Training App" reference, adapted for a Lifestyle Comparison Media.

## Concept
- **Goal**: Help users find the "correct" item to buy without regret.
- **Target**: People overwhelmed by choices, wanting objective advice.
- **Tone**: Clean, Trustworthy, Bright (White/Blue/Slate), "Pro" aesthetic.

## Section Breakdown

### 1. Hero (First View)
- **Copy**: "後悔しない「正解」を選ぶ。" (Choose the "correct" answer without regret.)
- **Sub-copy**: "情報過多の時代に、専門家と編集部が導く「ベストバイ」。"
- **Visual**: Bright, clean lifestyle imagery (not dark).
- **CTA**: Search Bar ("何をお探しですか？") + Popular Categories.
- **Note**: Remove "Self-funded" badge (User request).

### 2. Empathy / Problem (共感・問題提起)
- **Theme**: "Why shopping is stressful."
- **Points**:
  - "Reviews are contradictory." (口コミがバラバラ)
  - "Specs are confusing." (スペックが難解)
  - "Don't want to fail." (失敗したくない)
- **Design**: Clean white background, relatable icons.

### 3. Solution / Value (解決策・価値)
- **Theme**: "Why Best Buy Guide works."
- **Points**:
  - **Objective Verification**: Data-driven testing.
  - **Expert Eyes**: Professional evaluation.
  - **Comparison**: Clear "Winner" and "Loser".
- **Visual**: UI Mockup of a Comparison Table or Radar Chart.

### 4. Latest Reviews (機能・コンテンツ紹介)
- **Theme**: "Latest Verified Items."
- **Component**: Bento Grid (already implemented, keep it).
- **Content**: Robot Vacuum, Frying Pan, Washing Machine.

### 5. Categories (機能一覧)
- **Theme**: "Find by Category."
- **Design**: Pill-shaped or Card-shaped links.

### 6. Future / Benefit (ベネフィット)
- **Theme**: "Life with the Best Buy."
- **Copy**: "「いいモノ」は、毎日を変える。"
- **Visual**: Happy lifestyle image (Bright).

### 7. Final CTA
- **Copy**: "さあ、あなたの「正解」を見つけよう。"
- **Action**: Search Bar or Category Links again.

## Design Rules
- **Colors**: White background, Slate text, Blue primary. Avoid dark/black heavy sections.
- **Typography**: Sans-serif, readable, "Pro" feel.
- **Spacing**: Generous whitespace.
