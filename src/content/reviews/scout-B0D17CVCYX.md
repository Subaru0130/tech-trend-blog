# Sennheiser ACCENTUM True Wireless

Sennheiser ACCENTUM True Wirelessを実際に使用して検証しました。本当におすすめできるのか、正直にレビューします。


ノイズキャンセリングの強度（低音・人の声の遮断率）、音質のバランス、機能性（マルチポイント・外音取り込み）、装着感の観点で実機を検証しました。
結論から言うと、ゼンハイザーのフラッグシップ機譲りの音質特性を維持しつつ、実用十分なノイズキャンセリング性能を搭載した、音質重視のミドルレンジモデルとして優秀です。

### 1. 検証：ノイズキャンセリングの強度、音質のバランス、機能性、装着感での実力

ノイズキャンセリングの強度（低音・人の声の遮断率）
ハイブリッド方式のノイズキャンセリングを搭載していますが、強度はいわゆる「静寂」を作り出すトップクラス（BoseやSonyの最上位機）には及びません。
電車やバスの走行音（ゴーという低周波ノイズ）は7割から8割程度カットされ、音楽を再生すれば気にならなくなるレベルです。一方で、人の話し声や車内アナウンスといった中高音域のカット率は平均的です。無音状態でカフェの隣席の会話を完全に消すことは難しいため、あくまで「音楽への没入感を高めるための補助機能」と捉えるのが適切です。ただし、この価格帯としては十分な水準をクリアしています。

音質のバランス
本機の最大の強みです。上位機種と同等のトランスデューサー（ドライバー）を採用しており、ゼンハイザー特有の「広がり」と「分離感」のあるサウンドは健在です。
低音域は量感がありつつもボワつかず、タイトに鳴ります。中高音域も埋もれることなくクリアで、ボーカルの質感が生々しく再現されます。ドンシャリで誤魔化すのではなく、原音の雰囲気を損なわないバランスの良さは、同価格帯の製品群の中で頭一つ抜けています。

機能性（マルチポイント・外音取り込み・バッテリー）
2台同時接続のマルチポイントに対応しており、切り替えはスムーズです。
外音取り込み機能は、わずかにマイク特有の強調感（ホワイトノイズ等）を感じるものの、レジでの会話やアナウンスを聞く用途には支障ありません。
バッテリー性能はケース込みで最大28時間と標準的ですが、イヤホン単体での再生時間が長めに確保されているため、長時間の移動でもストレスなく使用できます。

装着感
エルゴノミックデザインを採用しており、耳のくぼみにフィットする形状です。筐体は極端に小さくはありませんが、耳への収まりが良く、頭を振ってもズレにくい安定感があります。この密閉度の高さが、パッシブ（物理的）な遮音性を高める役割も果たしています。

### 2. 競合製品との比較

比較対象である Technics EAH-AZ40M2 と比較した場合、本製品の立ち位置は以下の通りです。

音質：Sennheiser ACCENTUM True Wireless の勝利
Technics EAH-AZ40M2はLDACコーデックに対応しており、解像度の高いきめ細やかな音が特徴ですが、低音の迫力や音場の広がり（スケール感）に関しては、Sennheiserに分があります。音楽を「楽しむ」という点では本製品が優位です。

機能性：Technics EAH-AZ40M2 の勝利
Technics EAH-AZ40M2は「3台マルチポイント」に対応しており、PC、スマホ、タブレットを同時に待機させることが可能です。Sennheiserは一般的な2台接続に留まります。

装着感と携帯性：Technics EAH-AZ40M2 の勝利
Technicsは非常にコンパクトで、耳の穴が小さい人でも圧迫感が少ない設計です。Sennheiserも装着感は良好ですが、筐体のサイズ感とケースの携帯性ではTechnicsの方がコンパクトにまとまっています。

ノイズキャンセリング：引き分け（特性の違い）
どちらもハイエンド機ほどの強力な消音性能はありません。Sennheiserは筐体の厚みによる物理的な遮音性が高く、Technicsは筐体が軽いため長時間つけていても疲れにくいという違いがあります。実用レベルの消音能力としては同程度です。

### 3. 最終判定（買うべきか？）

### この製品がおすすめな人
*   音質を最優先したい人：3万円以下の予算で、オーディオブランドらしい厚みのある低音と広がりのあるサウンドを手に入れたいなら、迷わずこちらを選ぶべきです。
*   物理的な遮音性も重視する人：耳にしっかり詰まる装着感を好み、音楽を再生することで周囲の雑音をシャットアウトしたい通勤・通学ユーザーに適しています。
*   長時間の連続再生を求める人：イヤホン単体でのスタミナ性能が安定しています。

### おすすめできない人
*   3台以上のデバイスを使いこなす人：PC、私用スマホ、社用スマホなどをシームレスに切り替えたい場合は、3台マルチポイント対応の Technics EAH-AZ40M2 の方が利便性は圧倒的に上です。
*   耳の穴が非常に小さい人：本製品もフィット感は良いですが、耳が小さい女性や小柄な方には、より小型軽量な Technics EAH-AZ40M2 の方が圧迫感が少なく快適です。
*   LDACでのハイレゾ再生にこだわる人：Androidユーザーで、スペック上の高解像度転送を重視する場合も、LDAC対応のTechnicsが選択肢となります。
